# رابط خط فرمان (CLI)

Shekar یک رابط خط فرمان (CLI) برای پردازش سریع متن و تولید بصری‌سازی ارائه می‌دهد.  
می‌توانید متن فارسی را نرمال کنید یا مستقیماً از فایل/متن ورودی، ابرواژه بسازید.

**نحوهٔ استفاده**

```console
shekar [COMMAND] [OPTIONS]
```

### فرمان‌ها

1. `normalize`

برای نرمال‌سازی متن فارسی با یکدست‌سازی فاصله‌ها، نویسه‌ها و اعراب.  
این فرمان هم روی فایل و هم روی متن مستقیم کار می‌کند.

**گزینه‌ها**

- `-i, --input` مسیر فایل متنی ورودی  
- `-o, --output` مسیر ذخیرهٔ متن نرمال‌شده (در صورت عدم تعیین، خروجی در stdout چاپ می‌شود)  
- `-t, --text` دریافت متن مستقیم به‌جای فایل  
- `--encoding` تعیین اجباری کُدگذاری فایل ورودی  
- `--progress` نمایش نوار پیشرفت (به‌صورت پیش‌فرض فعال است)  

**نمونه‌ها**

<!-- termynal -->
```console
# نرمال‌سازی یک فایل و ذخیرهٔ خروجی
shekar normalize -i ./corpus.txt -o ./normalized_corpus.txt
```

</br>

<!-- termynal -->
```console
# نرمال‌سازی متن مستقیم
shekar normalize --text "درود پرودگار بر ایران و ایرانی"
```

1. `wordcloud`

برای تولید تصویر ابرواژه (PNG) از متن فارسی، از فایل یا متن مستقیم.  
پیش‌پردازش به‌صورت خودکار نشانه‌گذاری، اعراب، ایست‌واژه‌ها، نویسه‌های غیرفارسی را حذف می‌کند و فاصله‌گذاری را نرمال می‌سازد.

---

**گزینه‌ها**

- `-i, --input` فایل متنی ورودی  
- `-t, --text` دریافت متن مستقیم به‌جای فایل  
- `-o, --output` **(الزامی)** مسیر خروجی فایل PNG
- `--bidi` اعمال **bidi reshaping** برای نمایش صحیح متن فارسی (پیش‌فرض: `False`) 
- `--mask` ماسک شکل (`Iran`، `Heart`، `Bulb`، `Cat`، `Cloud`، `Head`) یا مسیر تصویر سفارشی  
- `--font` قلم مورد استفاده (`sahel`، `parastoo` یا مسیر TTF سفارشی)  
- `--width` عرض تصویر برحسب پیکسل (پیش‌فرض: 1000)  
- `--height` ارتفاع تصویر برحسب پیکسل (پیش‌فرض: 500)  
- `--bg-color` رنگ پس‌زمینه (پیش‌فرض: white)  
- `--contour-color` رنگ حاشیه (پیش‌فرض: black)  
- `--contour-width` ضخامت حاشیه (پیش‌فرض: 3)  
- `--color-map` نقشهٔ رنگ Matplotlib برای واژه‌ها (پیش‌فرض: Set2)  
- `--min-font-size` کمینهٔ اندازهٔ فونت (پیش‌فرض: 5)  
- `--max-font-size` بیشینهٔ اندازهٔ فونت (پیش‌فرض: 220)  

---

**نمونه‌ها**

<!-- termynal -->
```console
# تولید ابرواژه از فایل متنی
shekar wordcloud -i ./corpus.txt -o ./word_cloud.png
```

</br>

<!-- termynal -->
```console
# تولید ابرواژه از متن مستقیم با ماسک سفارشی

shekar wordcloud --text "درود پرودگار بر ایران و ایرانی" 
\ -o ./word_cloud.png --mask Heart
```

**نکته:** اگر حروف در ابرواژهٔ خروجی **از هم جدا** دیده می‌شوند، گزینهٔ `--bidi` را فعال کنید تا شکل‌دهی فارسی به‌درستی انجام شود.
